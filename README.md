# 문법Garden 🌱

A Korean grammar practice app — spinner wheel edition.

## What it does

- Randomly selects grammar points and sentences for practice
- Single-file PWA-ready web app
- Works offline after first load

## Live app

**[https://jahnielkr.github.io/munbeop-garden/](https://jahnielkr.github.io/munbeop-garden/)**

## Usage

Open `index.html` in any modern browser, or visit the GitHub Pages URL above.
